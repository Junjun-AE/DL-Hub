# 🧠 DL-Hub — 深度学习任务管理平台

> 📖 完整文档请查看 [README.md](./README.md)（中英双语）

## 一句话介绍

DL-Hub 是一个一站式深度学习平台，在统一的 Web 管理界面下集成了**六大计算机视觉任务**（分类/检测/分割/异常检测/OCR/工业缺陷检测），覆盖从数据准备、模型训练到格式转换、部署打包的完整流水线。

## 快速启动

```bash
pip install -r requirements.txt
python run_dlhub.py
```

浏览器自动打开 `http://localhost:7860`，首次使用需注册账号。
