功能概览

DL-Hub · 工业缺陷检测

SevSeg-YOLO

检测 + 评分 + 分割 三合一工业缺陷检测系统

🔍 缺陷定位
📊 严重程度评分 0-10
🎭 零标注近似分割
⚡ TensorRT ~6ms

SevSeg-YOLO 是 DL-Hub 平台的工业缺陷检测模块。传统方案需要三个独立模型分别完成检测、评分、分割；SevSeg-YOLO 在单次前向传播中同时完成这三个任务，仅需框标注 + severity 评分字段，无需像素级分割标注。

💡
核心创新：Score Head 直接集成到 YOLO 检测架构中输出严重程度评分，MaskGenerator 从检测框推导像素级分割结果，实现了「一个模型解决三个问题」的效率突破。

Score Head 仅增加了极少的参数量（约 0.1M），不影响检测速度。输出的 severity 值是一个 0-10 的连续浮点数，代表缺陷的严重程度。

三、Gaussian NLL 损失函数

评分回归使用高斯负对数似然损失（Gaussian Negative Log-Likelihood），而非简单的 MSE。

🔬
为什么不用 MSE？
MSE 假设所有样本的预测不确定性相同。而实际中：严重缺陷（severity 8-10）的边界比轻微缺陷（severity 1-2）更模糊。Gaussian NLL 允许模型同时学习预测值和预测不确定性。
Gaussian NLL Loss 公式
L = ½ × [ log(σ²) + (y - μ)² / σ² ]

μ (mu) = 模型预测的 severity 值

σ (sigma) = 不确定性标准差（由 gaussian_sigma 配置，默认 0.10）

y = 标注的真实 severity 值

训练技巧

  • 冷启动：使用 YOLO26m.pt 初始化检测权重，Score Head 随机初始化
  • score_loss 不下降：尝试降低学习率到 0.005 或增大 lambda_score

MaskGenerator 原理

MaskGenerator 不需要任何分割标注，利用 YOLO 检测网络自身的 FPN 特征图（P3/P4/P5）推导像素级缺陷 Mask。核心洞察：检测模型的 FPN 特征天然编码了「缺陷 vs 正常」的语义信息——缺陷区域在特征图上呈现低激活（暗色），正常区域呈现高激活（亮色）。

MaskGenerator 完整处理流程（6步）
Step 1 尺度自适应特征选择

根据模型规模自动选择最佳特征组合：
• 小模型（n/s，通道数≤128）：Layer2 + P3 + P4，权重 [0.35, 0.35, 0.30]
• 大模型（m/l/x）：P3 + P4 + P5，权重 [0.45, 0.35, 0.20]
P3 分辨率最高（stride=8），提供空间细节；P5 分辨率最低（stride=32），提供语义全局信息。

Step 2 双峰通道选择 + L2 激活 核心改进

对每个尺度的特征，在 bbox 对应的区域内裁剪，然后选出最能区分缺陷的通道：

双峰通道选择算法：
① 对 bbox crop 内每个通道的所有像素值排序
② 计算 最暗 30% 像素的均值（代表缺陷区域）和 最亮 30% 像素的均值（代表正常区域）
③ 两者之差 = 双峰间距（bimodal gap）
④ 选双峰间距最大的 Top-K 个通道（默认 K=48）
⑤ 在选出的通道上计算 L2 范数：activation = √(Σ channel_i²)
⑥ min-max 归一化到 [0, 1]
💡
为什么双峰比方差好？方差最大的通道可能只是「背景纹理变化大」，而双峰间距最大的通道是「bbox 内存在两类明显不同区域」——这正是缺陷和正常区域的分离信号。
Step 3 多尺度加权融合

将各尺度的激活图上采样到最高分辨率（P3 级别），按预设权重加权平均。多尺度融合的好处：P3 提供精确的空间位置，P4/P5 提供稳健的语义判断，综合后比单一尺度更可靠。

Step 4 Canny 边缘引导上采样

将低分辨率激活图（~5×7 像素）上采样到 bbox 实际像素大小（如 80×60），同时利用原图的边缘信息引导上采样质量：

① 双线性插值上采样到目标尺寸
② 对原图 bbox 区域做 Canny 边缘检测 → 二值边缘图
③ 膨胀边缘 → 创建「边缘区」和「平滑区」
④ 边缘区：保留原始上采样值（保持缺陷边界锐利）
⑤ 平滑区：施加高斯模糊（抑制插值噪声）
⑥ 混合：result = raw × edge_mask + blurred × (1 - edge_mask)
Step 5 自适应二值化

将连续激活值 [0, 1] 转换为二值 mask {0, 1}。使用 cv2.adaptiveThreshold（高斯加权，blockSize=15, C=-5），自动适应局部亮度变化。对于太小的 bbox（边长<15px）退化为 Otsu 全局阈值。

Step 6 形态学后处理

闭操作（7×7 椭圆核）：填充 mask 内的小孔洞，连接近邻区域
开操作（3×3 椭圆核）：去除孤立的噪声小点
最终输出 {0, 1} 二值 mask，1 = 缺陷像素，0 = 正常像素。

📐 坐标系统与位置还原
整个 MaskGenerator 在模型输入空间（640×640，含 letterbox 填充）中工作，不会产生位置偏移：

① YOLO 输出原图坐标 bbox → model.py 映射到 640×640 模型空间 bbox_model
② MaskGenerator 在 P3/P4/P5 上按 bbox_model 裁剪 → 生成 mask_raw（模型空间大小）
③ model.py 将 mask_raw resize 回原图 bbox 大小 → 最终 mask

示例：原图 2000×976，bbox 原图坐标 (500,400)-(600,450) → 模型空间 (160,292)-(192,308) → P3 裁剪 3×5 → 上采样到 16×32 → resize 回 100×50 = 原图 bbox 大小 ✓
💡
核心思想：检测模型的 FPN 特征图天然编码了缺陷语义——缺陷区域在高方差/高双峰通道上呈现独特的激活模式。MaskGenerator 利用这一特性，通过通道选择 + 多尺度融合 + 边缘引导上采样，将特征图的语义信息转化为像素级 mask。这是零标注的近似方法，精度取决于缺陷与背景的特征差异程度。

常见问题

Q: severity 不填会怎样？
默认 5.0。强烈建议手动填写准确值。
Q: 可以不使用评分功能吗？
可以。所有样本 severity 设为相同值，模型退化为普通检测器。
Q: MaskGenerator Mask 精度如何？
MaskGenerator 从 FPN 特征图推导 mask，精度取决于缺陷与背景在特征空间中的分离度。边缘清晰、与背景对比明显的缺陷效果好；与背景纹理相似的模糊缺陷可能不够精确。这是零标注的近似方法，如需像素级精确分割，请使用语义分割模块。
Q: 为什么 MixUp 必须关闭？
MixUp 混合两张图片像素和标签。severity 不能线性插值——「严重裂纹×0.3 + 轻微划痕×0.7 = severity 5.2」没有物理意义。
Q: 支持哪些 YOLO 版本？
推荐 YOLO26-Score（最新架构）。也支持 YOLO11-Score。需要 ultralytics ≥ 8.4.0。
